Usage: oci session export [OPTIONS]

  Exports a CLI session into a zip that can be imported. The session exported
  is the active profile (specified either by defaults or the --config-file /
  --profile arguments.

Options:
  --output-file PATH  The name of the export file to create, including the path,
                      minus any format-specific extension. [required]
  -?, -h, --help      For detailed help on any of these individual commands,
                      enter <command> --help.
  --force             If the ouptut zip already exists, overwrite the existing
                      zip without a confirmation prompt.
